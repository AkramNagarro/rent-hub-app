# RentHubApp

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 21.1.3.

# Project is divided into two layers

1. Frontend - This project's front end part (rent-hub-app) is deployed on vercel free host server

2. Backend - Project's backend part (backend) is deployed on render free host server. Backend is implement using JSON Server (db.json).
